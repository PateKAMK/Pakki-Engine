
# PakkiEngine Github pages

**Welcome!**

## What is PakkiEngine?

**PakkiEngine** is simple 2d game-engine which is developed with **c++** using **opengl 3+** rendering.
**Pakki** is done as part of school project and will be developed further in future courses.

## Features
- spritebatching
- audio
- android support
- PakkiGUI
- Lua scripting
